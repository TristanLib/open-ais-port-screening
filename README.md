# Open AIS Port Screening public reproducibility package

This code-only archive contains analysis source, fixed public configurations,
review-v10 regression tests, a synthetic smoke sample, the frozen method
protocol, exact public-data reproduction commands, and sanitized two-waterway
Web demonstration layers including de-identified encounter evidence cards.

The package intentionally excludes manuscripts, abstracts, reviewer material,
submission correspondence, raw or processed full-resolution AIS, identity-
bearing analytical tables, author contact details, local paths, environments,
screenshots, and generated delivery packages.

The outputs are historical candidate-screening evidence for qualified human
review. They are not accident probabilities, near-miss labels, navigational
advice, enforcement findings, or a real-time VTS service.

Canonical public repository:
<https://github.com/TristanLib/open-ais-port-screening>

Some regression-test filenames retain `review_v9` for path compatibility; their
assertions and expected schemas are updated for review-v10.

Start with `docs/REPRODUCIBILITY.md` and `docs/PUBLIC_DATA_BOUNDARY.md`.
